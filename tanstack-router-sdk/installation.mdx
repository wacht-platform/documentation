## Requirements

- `@tanstack/react-router` 1+
- React 18+

## Install

<CodeGroup>

```bash pnpm
pnpm add @wacht/tanstack-router @wacht/jsx @wacht/types @tanstack/react-router
```

```bash npm
npm install @wacht/tanstack-router @wacht/jsx @wacht/types @tanstack/react-router
```

</CodeGroup>

## Environment

```bash
WACHT_PUBLISHABLE_KEY=pk_test_xxx
```

Use `WACHT_API_KEY` only when calling backend APIs from server runtime.
