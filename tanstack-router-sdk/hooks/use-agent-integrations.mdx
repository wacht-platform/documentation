# useAgentIntegrations

This hook is stale and should not be used.

The older frontend integration routes are no longer the active contract.

Use `useActorMcpServers` instead for the current actor-scoped MCP connection surface:

- list actor MCP servers
- start actor MCP connect flow
- disconnect actor MCP servers

The current frontend runtime API is documented in:

- `/guides/ai-agents/frontend-api-runtime-routes`
