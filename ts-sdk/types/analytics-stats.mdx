---
title: 'Analytics Stats'
description: 'Represents analytics statistics in Wacht'
---

This interface defines the structure of analytics statistics data.

```typescript
export interface AnalyticsStats {
    total_users?: number;
    active_users_30d?: number;
    new_users_30d?: number;
    total_organizations?: number;
    total_workspaces?: number;
    ai_agents_count?: number;
    ai_workflows_count?: number;
    ai_executions_30d?: number;
    knowledge_base_documents?: number;
    storage_used_gb?: number;
}
```

#### Properties

- `total_users` (number, optional): Total number of users.
- `active_users_30d` (number, optional): Number of active users in the last 30 days.
- `new_users_30d` (number, optional): Number of new users in the last 30 days.
- `total_organizations` (number, optional): Total number of organizations.
- `total_workspaces` (number, optional): Total number of workspaces.
- `ai_agents_count` (number, optional): Number of AI agents.
- `ai_workflows_count` (number, optional): Number of AI workflows.
- `ai_executions_30d` (number, optional): Number of AI executions in the last 30 days.
- `knowledge_base_documents` (number, optional): Number of knowledge base documents.
- `storage_used_gb` (number, optional): Storage used in GB.
