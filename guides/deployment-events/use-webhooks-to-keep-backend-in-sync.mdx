# Use Webhooks to Keep Your Backend in Sync

Every webhook delivery uses this envelope:

```json
{
  "type": "<event_name>",
  "timestamp": "2026-03-04T10:00:00.000Z",
  "data": {}
}
```

`data` is the delivered event payload.
