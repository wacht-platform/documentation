# Introduction to Wacht Platform

Welcome to the Wacht Platform documentation. This section covers the **Backend API** and **Rust SDK** for building enterprise applications with authentication, user management, organizations, and AI tools.

## What is Wacht?

Wacht is a development platform that helps you build enterprise applications faster with:

- **Authentication & Authorization** - Email/password, phone OTP, magic links, social OAuth, MFA
- **Multi-Tenant Architecture** - Organizations, workspaces, roles, and permissions
- **User Management** - Complete user CRUD with emails, phones, and social connections
- **AI Tools** - AI agents with knowledge base integration, tools, and execution contexts
- **Analytics** - Real-time metrics with ClickHouse
- **Enterprise Features** - SSO, SCIM, domains, custom branding
- **Developer Tools** - Webhooks, API keys, segments, notifications
