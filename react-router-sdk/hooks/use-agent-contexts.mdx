# useAgentContexts

This hook is stale and should not be used.

The older `/agent/contexts` API family no longer exists.

Use the current runtime hooks instead:

- `useActorProjects` for actor-scoped projects
- `useProjectThreadFeed` or `useProjectThreads` for project threads
- `useProjectTasks` for board items
- `useProjectTaskBoardItem` for board-item detail, events, assignments, and filesystem
- `useAgentThreadConversation` for thread messages and execution

The current frontend runtime API is documented in:

- `/guides/ai-agents/frontend-api-runtime-routes`
