## Core Package

Install the main Wacht React SDK:

<CodeGroup>
```bash npm
npm install @snipextt/wacht
```

```bash yarn  
yarn add @snipextt/wacht
```

```bash pnpm
pnpm add @snipextt/wacht
```
</CodeGroup>

## Platform Adapters

Install the appropriate adapter for your React framework:

### Next.js

<CodeGroup>
```bash npm
npm install @snipextt/wacht-nextjs
```

```bash yarn
yarn add @snipextt/wacht-nextjs
```

```bash pnpm
pnpm add @snipextt/wacht-nextjs
```
</CodeGroup>

### React Router

<CodeGroup>
```bash npm
npm install @snipextt/wacht-react-router
```

```bash yarn
yarn add @snipextt/wacht-react-router
```

```bash pnpm
pnpm add @snipextt/wacht-react-router
```
</CodeGroup>

## Requirements

- **React** 18.0 or later
- **TypeScript** 4.5 or later (recommended)
- **Node.js** 16.0 or later

## Verification

Verify your installation by importing the core package:

```tsx
import { DeploymentProvider } from '@snipextt/wacht'

// If this import works without errors, you're ready to go!
```
