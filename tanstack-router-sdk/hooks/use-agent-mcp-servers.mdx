# useAgentMcpServers

`useAgentMcpServers` is removed.

The current frontend AI runtime exposes actor-scoped MCP connection routes, and the TanStack Router SDK hook for that surface is `useActorMcpServers`.

Use this instead:

- [useActorMcpServers](./use-actor-mcp-servers)
- [useAgentSession](./use-agent-session)
