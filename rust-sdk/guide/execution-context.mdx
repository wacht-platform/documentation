# AI Execution Context API

This API surface is removed.

The backend AI contract no longer uses execution contexts as a first-class route family.

Use the current backend runtime administration model instead:

- actor projects
- actor project threads
- actor MCP servers

See:

- `/guides/ai-agents/backend-api-admin-routes`
- `/rust-sdk/guide/agents`
