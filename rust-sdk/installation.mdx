## Add dependency

<CodeGroup>

```toml Cargo.toml
[dependencies]
wacht = "0.1.0-beta.1"
tokio = { version = "1", features = ["macros", "rt-multi-thread"] }
```

```toml With Axum middleware
[dependencies]
wacht = { version = "0.1.0-beta.1", features = ["axum"] }
tokio = { version = "1", features = ["macros", "rt-multi-thread"] }
```

</CodeGroup>

## Features

- `axum` (default): enables `AuthLayer`, permission layers, and extractors.

## Minimum runtime expectations

- Rust stable toolchain
- Tokio runtime
