Welcome to the Wacht platform documentation. 

Wacht is a comprehensive infrastructure tailored for modern developers, providing everything you need to build scalable, multi-tenant applications. We offer robust primitives for authentication, organization management, API authorization, and embedded AI agents.

Whether you are building a new React application from scratch or integrating powerful backend services with Rust, our SDKs and APIs are designed to be fast, secure, and developer-friendly.
