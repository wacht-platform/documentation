---
title: 'Organization Role'
description: 'Represents an Organization Role in Wacht'
---

This interface defines the structure of an Organization Role object.

```typescript
export interface OrganizationRole {
    id?: string;
    name?: string;
    description?: string;
    permissions?: string[];
    is_default?: boolean;
    created_at?: string;
}
```

#### Properties

- `id` (string, optional): The unique identifier of the organization role.
- `name` (string, optional): The name of the organization role.
- `description` (string, optional): A description of the organization role.
- `permissions` (string[], optional): A list of permissions associated with the role.
- `is_default` (boolean, optional): Indicates if this is a default role.
- `created_at` (string, optional): The timestamp when the organization role was created.
