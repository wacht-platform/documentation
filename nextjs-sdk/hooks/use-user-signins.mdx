## Purpose

`useUserSignins()` lists the current account's sign-in records and exposes a helper to revoke one sign-in.

It is implemented in the shared JSX layer and re-exported by the framework packages.

## Import

```tsx
import { useUserSignins } from '@wacht/nextjs'
```

## Current return shape

```ts
{
  signins: SignIn[] | undefined
  error: Error | null
  removeSignin: (id: string) => Promise<ApiResult<unknown>>
  refetch: () => Promise<SignIn[] | undefined>
  loading: boolean
}
```

## Returned fields

<ResponseField name="signins" type="SignIn[] | undefined">
  The sign-in records currently returned by `GET /me/signins`.
</ResponseField>

<ResponseField name="error" type="Error | null">
  The current SWR error for the sign-in list.
</ResponseField>

<ResponseField name="loading" type="boolean">
  `true` while the client is still loading or the sign-in list request is still in flight.
</ResponseField>

<ResponseField name="removeSignin" type="(id: string) => Promise<ApiResult>">
  Revokes one sign-in by calling `POST /me/signins/{id}/signout`.
</ResponseField>

<ResponseField name="refetch" type="() => Promise<void>">
  Revalidates the sign-in list.
</ResponseField>

## Route contract used by this hook

The current implementation uses:

- `GET /me/signins`
- `POST /me/signins/{id}/signout`

## Example

```tsx
import { useUserSignins } from '@wacht/nextjs'

export function ActiveSessions() {
  const { signins, loading, error, removeSignin, refetch } = useUserSignins()

  if (loading) return <div>Loading sign-ins...</div>
  if (error) return <div>Failed: {error.message}</div>

  return (
    <ul>
      {(signins ?? []).map((signin) => (
        <li key={signin.id}>
          <span>{signin.client_name ?? signin.id}</span>
          <button
            onClick={async () => {
              await removeSignin(signin.id)
              await refetch()
            }}
          >
            Revoke
          </button>
        </li>
      ))}
    </ul>
  )
}
```
