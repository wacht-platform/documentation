---

## Purpose

`useActiveTenancy()` derives the active organization membership and active workspace membership from the current session.

It does not fetch active tenancy from a dedicated endpoint.

Instead, it combines:

- `useSession()`
- `useOrganizationMemberships()`
- `useWorkspaceMemberships()`

## Import

```tsx
import { useActiveTenancy } from '@wacht/react-router'
```

## Current return shape

```ts
{
  loading: boolean
  orgMembership: OrganizationMembership | null
  workspaceMembership: WorkspaceMembership | null
}
```

## Returned fields

<ResponseField name="loading" type="boolean">
  `true` while the session, organization memberships, or workspace memberships are still loading.
</ResponseField>

<ResponseField name="orgMembership" type="OrganizationMembership | null">
  The organization membership whose ID matches `session.active_signin?.active_organization_membership_id`.
</ResponseField>

<ResponseField name="workspaceMembership" type="WorkspaceMembership | null">
  The workspace membership whose ID matches `session.active_signin?.active_workspace_membership_id`.
</ResponseField>

## How it resolves active tenancy

Current implementation rules:

- if there is no session, no sign-ins, or no active sign-in, both values resolve to `null`
- `orgMembership` is selected by matching the current session's active organization membership ID against the organization membership list
- `workspaceMembership` is selected by matching the current session's active workspace membership ID against the workspace membership list

This hook is a client-side composition helper. It does not own the active tenancy state itself.

## Example

```tsx
import { useActiveTenancy } from '@wacht/react-router'

export function ActiveTenancySummary() {
  const { loading, orgMembership, workspaceMembership } = useActiveTenancy()

  if (loading) return <div>Loading tenancy...</div>

  return (
    <div>
      <p>Organization: {orgMembership?.organization?.name ?? 'None'}</p>
      <p>Workspace: {workspaceMembership?.workspace?.name ?? 'None'}</p>
    </div>
  )
}
```
