# Signature Verification with Raw Body

Wacht follows the [Standard Webhooks spec](https://github.com/standard-webhooks/standard-webhooks?tab=readme-ov-file).

Use a prebuilt Standard Webhooks library for verification.

```ts
import { Webhook } from "standardwebhooks";

const wh = new Webhook(base64_secret);
wh.verify(webhook_payload, webhook_headers);
```

Headers sent by Wacht:

1. `webhook-id`
2. `webhook-timestamp`
3. `webhook-signature`
