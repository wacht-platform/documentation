## Requirements

- React 18+
- React Router 7+

## Install

<CodeGroup>

```bash pnpm
pnpm add @wacht/react-router @wacht/jsx @wacht/types react-router
```

```bash npm
npm install @wacht/react-router @wacht/jsx @wacht/types react-router
```

</CodeGroup>

## Environment

```bash
WACHT_PUBLISHABLE_KEY=pk_test_xxx
```

- Publishable key: required for auth/session derivation
- API key: only needed for backend API access from server
